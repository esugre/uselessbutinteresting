# Useless but Interesting

**Useless but Interesting** ist ein kleines, statisches Webprojekt, das eine Sammlung von „unnützen, aber schönen Dingen“ präsentiert. Es dient als eine Art Mini-Museum für Kuriositäten, die keinen praktischen Nutzen haben, aber trotzdem faszinieren.

## Inhalte

* Kartenbasierte Darstellung einzelner Objekte
* Kurze Texte zu jedem Objekt
* Persönliche Begründungen im Sinne von: *„Warum fasziniert mich das?“*
* Fokus auf minimalistisches Layout und klare Struktur

## Ziel des Projekts

Dieses Projekt entstand als leichtgewichtige Übung, um statische Webseiten zu gestalten und Content visuell ansprechend aufzubereiten. Es verbindet HTML/CSS-Grundlagen mit konzeptionellem Arbeiten.

## Tech-Stack

* Reines **HTML**
* **CSS** für Layout & Gestaltung
* Kein JavaScript, keine Frameworks

## Status

Abgeschlossenes bzw. fortlaufend erweiterbares Hobbyprojekt. Neue Kuriositäten können jederzeit ergänzt werden.
